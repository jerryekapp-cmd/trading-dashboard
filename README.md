# Trading Dashboard

Web-based stock analysis dashboard built with PHP and TradingView.

## Features
- Multi-chart layout
- Multi-timeframe (1m, 5m, 30m, daily)
- Technical indicators (SMA, RSI)
- Internal analysis tool

## Tech Stack
- PHP 8.x
- JavaScript
- TradingView widgets

## Status
In development (internal use, expanding features)
